STIMA Readme

Introduction

software to collect weather data contributed by citizens; to make these data available for weather services and homeland security; to provide feedback to the data contributors so that they have the tools to check and improve their data quality

http://rmap.cc

Howto

STIMA Report over Ethernet:

1)

open sketch arduino/sketchbook/rmap/rmap/rmap.ino

2)

in rmap-config.h set STIMA_MODULE_TYPE_REPORT_ETH in MODULE_VERSION define

3)

open arduino/sketchbook/library/RmapConfig/sensors_config.h and set true or false sensors's define and json's define in order to enable or disable relative sensor driver and library

STIMA Report over GSM:

1)

open sketch arduino/sketchbook/rmap/rmap/rmap.ino

2)

in rmap-config.h set STIMA_MODULE_TYPE_REPORT_GSM in MODULE_VERSION define

3)

open arduino/sketchbook/library/RmapConfig/sensors_config.h and set true or false sensors's define and json's define in order to enable or disable relative sensor driver and library
